# Employee Payroll System

## 📌 Overview

Employee Payroll System is a desktop application developed using **Python** and **Tkinter**. It helps manage employee information, calculate salaries based on leave records, generate payslips, and search employee details through a user-friendly graphical interface.

---

## ✨ Features

- 👤 Employee Registration
- 🔐 Employee Login
- 🔍 Search Employee Details
- 📋 Display All Employee Records
- 💰 Net Salary Calculation
- 🧾 Pay Slip Generation
- 💾 Local Data Storage using Pickle

---

## 🛠️ Technologies Used

- Python
- Tkinter
- Object-Oriented Programming (OOP)
- Pickle (Data Storage)

## 🚀 How to Run

1. Install Python 3.x.
2. Download or clone this repository.
3. Open the project folder.
4. Run the application:

```bash
python payroll_system.py
```
